Galaga
============

<a href="http://edwardl.net:3001">Demo</a>

#Synopsis
Galaga is a shooter arcade game. The player can control a spaceship and enemies appear from every direction. The player earns gold by destroying the enemies. The gold is used to upgrade the playerâ€™s spaceships and purchase better performance spaceships.

#Controls
<ul>
  <li><b>W</b>: Moves the ship forward</li>
  <li><b>S</b>: Moves the ship backward</li>
  <li><b>A</b>: Moves the ship left</li>
  <li><b>D</b>: Moves the ship right</li>
  <li><b>Move Mouse</b>: Moves the aim cursor</li>
</ul>
#Browser Compatibility
Tested on Chrome and Firefox on Windows and OS X

#Technology Used
<ul>
  <li>Nodejs</li>
  <li>Expressjs</li>
  <li>MongoDB</li>
  <li>Createjs</li>
</ul>
